静态分析工具是指，在不编译或执行代码的情况下，对代码进行分析的工具。你的编译器就是其中一种，也是第一道防线。

许多此类工具是免费的，有些则对开源项目免费。

cppcheck 和 clang-tidy 是两个广受欢迎且免费的工具，其与主流 IDE 和编辑器有良好的集成支持。

\begin{myTip}{练习：启用更多静态分析}
Visual Studio 用户：研究一下随 Visual Studio 附带的 Microsoft 静态分析器。考虑使用 Clang Power Tools，并下载 cppcheck 的 Visual Studio 插件。

CMake 用户：启用 cppcheck 和 clang-tidy 的集成。
\end{myTip}

\begin{myTip}{练习：在 IDE 中启用静态分析}
大多数现代 IDE 都内置了对 clang-tidy 和其他静态分析工具的支持。请研究如何启用这些功能，并配置你的 .clang-tidy 文件，以开启适合你项目的分析检查。
\end{myTip}
