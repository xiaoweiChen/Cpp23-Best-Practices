果你想在自己所从事的领域变得更好，就必须持续不断地学习，而且你身边有很多资源可以继续你的 C++ 教育。

\mySubsectionNoFile{63.1}{知道如何提问}

Kate Gregory 发表了一篇关于如何提问的优秀文章\footnote{\url{http://www.gregcons.com/KateBlog/HowToAskForCCodingHelp.aspx}}。其中一些关键要点是：

\begin{itemize}
\item
不要使用截图

\item
使用有意义的变量名

\item
添加一些测试代码

\item
认真倾听别人给你的建议
\end{itemize}

\mySubsectionNoFile{63.2}{会议和本地用户组}

在你附近几乎肯定有一个。这是建立人脉和学习新知识的好方法。请查看 ISO C++ 全球会议列表\footnote{\url{https://isocpp.org/wiki/faq/conferences-worldwide}} 和 Meeting C++ 的用户组列表\footnote{\url{https://meetingcpp.com/usergroups/}}。

我写完这本书时，正值全球 COVID-19 大流行期间。因此，目前会议和用户组活动大多暂停了。但这为许多人带来了新的机遇。

许多会议和用户组现在都转为线上举行。我们现在可以参加彼此的用户组活动了。例如，北丹佛都会区 C++ 聚会(North Denver Metro C++ Meetup)\footnote{\url{https://www.meetup.com/North-Denver-Metro-C-Meetup/}} 每月都有来自泰国的参与者。

\begin{myTip}{}
作者注：当你与 C++ 社区互动时，请记住要尊重他人，保持尊严；保持耐心；花时间去了解你所参与的特定社区的规则和规范。
\end{myTip}
