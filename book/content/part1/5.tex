
AI 编程助手正变得无处不在，几乎每个集成开发环境(IDE)和工具都内置了类似功能。它们看起来相当强大，能够生成令人信服的结果。然而，这些结果并不一定正确。因此，我建议你遵循以下最佳实践来使用 AI 编程助手：

\begin{enumerate}
\item
务必仔细检查你得到的结果。

\item
主要将其用作一个“智能橡胶鸭”(注：指通过向其解释问题来理清思路)。

\item
务必仔细检查你得到的结果。

\item
用它来“降低学习曲线”。如果你让 AI 生成某个特定技术或 API 用法的示例，你很可能会得到一个有意义的答案，但其中也可能包含一些错误。

\item
务必仔细检查你得到的结果。
\end{enumerate}
