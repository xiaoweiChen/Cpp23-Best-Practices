考虑到 C++ 并非一门纯粹的面向对象语言，你必须掌握许多不同的技术才能充分发挥 C++ 的潜力。

以下练习将帮助你接触其他编程语言。但事实上，目前很少有语言是纯粹的单范式语言。每种语言都有自己惯用的编程方式，这些方式都存在于该语言所偏爱的编程范式之内。

Ben Deane 推荐所有程序员都应该学习以下几种语言\footnote{\url{http://www.elbeno.com/blog/?p=420}}：

\begin{itemize}
\item
ALGOL 家族(C 及其衍生语言)

\item
Forth

\item
Lisp 及其方言

\item
Haskell

\item
Smalltalk

\item
Erlang
\end{itemize}

\begin{myTip}{练习：选择一门函数式语言来学习}
你能找到一门纯函数式语言吗？
\end{myTip}

\begin{myTip}{练习：选择一门面向对象语言来学习}
找到一门纯面向对象的语言甚至更难！就连如今的 Java 也有了 lambda 表达式。
\end{myTip}

\begin{myTip}{练习：选择一门语法不同的语言}
看起来像 C 语言家族的语言对你来说可能会更顺手。试着找一门外观截然不同的语言，以拓展你的思维。
\end{myTip}
