C++ 标准库在持续增长。在 C++23 标准中，标准库部分约占 1403 页，占整个标准的 66\%。而在 C++17 中，它只有 965 页。

了解标准库中发生的变化非常重要。本书只会涉及其中一部分内容，但 C++20 和 C++23 的一些亮点包括：

\begin{itemize}
\item
大幅扩展的日期/时间库(包含日历功能)

\item
对并发和线程支持的大量改进

\item
std::format

\item
ranges(范围)

\item
concepts(概念)

\item
<stacktrace> 和其他诊断辅助工具
\end{itemize}

\begin{myTip}{练习}
浏览 C++23 标准库的变更列表，看看哪些内容引起了你的注意：\url{https://en.cppreference.com/w/cpp/compiler_support/23}
\end{myTip}
